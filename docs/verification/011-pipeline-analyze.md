## 验证内容

QSNR 和 MSE 公式对已知 fp32 / quantized 张量对的正确性。

## 公式

```
QSNR = 10 * log10(||fp32||² / ||fp32 - quant||²)   [dB]
MSE  = mean((fp32 - quant)²)
```

其中 ||·||² 是 Frobenius norm（所有元素的平方和）。

## 给定数据

使用 Test 002 (int8 per_channel) 和 Test 007 (Linear forward) 的结果：

```python
x   = [[0.5, -0.25], [1.0, 0.75]]

y_q = [[0.12151, 0.67727, 1.37813], [2.59219, 6.19219, 9.8]]
```

## 手工推导

### QSNR(x, x_q)

```
||x||² = 0.5² + (-0.25)² + 1.0² + 0.75² = 0.25 + 0.0625 + 1.0 + 0.5625 = 1.875
||x - x_q||² = 0² + (-0.25+0.24609)² + 0² + 0² = 0.00001526...

QSNR = 10 * log10(1.875 / 0.00001526) ≈ 50.89 dB
```

### QSNR(W, W_q)

```
||W||² = 1²+2²+3²+4²+5²+6² = 1+4+9+16+25+36 = 91.0
||W - W_q||² = (1-1.0156)²+(2-1.9688)²+... = 0.003173...

QSNR ≈ 44.57 dB
```

### QSNR(y, y_q)

```
||y||² = 0.01 + 0.49 + 1.69 + 6.76 + 38.44 + 96.04 = 143.43
||y - y_q||² = ... = 0.007205...

QSNR ≈ 42.99 dB
```
