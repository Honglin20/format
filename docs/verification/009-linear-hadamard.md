# 009: Hadamard 自逆性验证

**对应测试函数**: `test_hadamard_self_inverse()`
**验证层级**: Layer 2 — 算子 + Transform

## 验证内容

Fast Walsh-Hadamard Transform (FWHT) 的 self-inverse 性质: `H(H(x)) == x`

## 量化公式（FWHT, last dim, size=2）

对于 last_dim=2（已是 2 的幂），FWHT 用 butterfly 算法：

```
H = 1/sqrt(2) * [[1, 1], [1, -1]]

H(x) = x @ H^T  (沿 last dim 变换)
```

对于 2D 张量 x(2,2)，沿 dim=1 做变换：
- x 的每一行独立变换
- H 矩阵: [[1/sqrt(2), 1/sqrt(2)], [1/sqrt(2), -1/sqrt(2)]]

## 给定数据

```python
x = [[0.5, -0.25], [1.0, 0.75]]
```

## 手工推导

1/sqrt(2) ≈ 0.70710678

### hadamard(x) — 逐行计算

**Row 0** [0.5, -0.25]:
```
h = 1: sum = 0.5+(-0.25)=0.25, diff = 0.5-(-0.25)=0.75
     /sqrt(2) → [0.25/1.4142, 0.75/1.4142] = [0.17678, 0.53033]
```

**Row 1** [1.0, 0.75]:
```
h = 1: sum = 1.0+0.75=1.75, diff = 1.0-0.75=0.25
     /sqrt(2) → [1.75/1.4142, 0.25/1.4142] = [1.23744, 0.17678]
```

H(x) = [[0.17678, 0.53033], [1.23744, 0.17678]]

### hadamard(hadamard(x)) — 自逆

**Row 0** [0.17678, 0.53033]:
```
sum = 0.17678+0.53033 = 0.70711, diff = 0.17678-0.53033 = -0.35355
/sqrt(2) → [0.70711/1.4142, -0.35355/1.4142] = [0.5, -0.25] ✓
```

**Row 1** [1.23744, 0.17678]:
```
sum = 1.23744+0.17678 = 1.41422, diff = 1.23744-0.17678 = 1.06066
/sqrt(2) → [1.41422/1.4142, 1.06066/1.4142] = [1.0, 0.75] ✓
```

H(H(x)) = [[0.5, -0.25], [1.0, 0.75]] = x ✓

> **float32 精度说明**: 由于 `1/sqrt(2)` 乘两次引入 ~6e-8 的浮点舍入误差，`H(H(x))` 与 `x` 不是 bit-exact 相等（差 1-2 ULP）。验证脚本对此断言使用 `torch.allclose(atol=1e-7)`，数学性质在实数域严格成立。
