## 手工推导

### Step 1: Quantize input（已验证于 002）

x_q = int8 per_channel(axis=0) of x
```
col0 amax=1.0: [0.5, 1.0] → int8 → [0.5, 1.0] → *1.0 = [0.5, 1.0]
col1 amax=0.75: [-0.25, 0.75] → norm=[-0.333, 1.0] → int8=[-0.328125, 1.0] → *0.75 = [-0.24609375, 0.75]
```
x_q = [[0.5, -0.24609375], [1.0, 0.75]]

### Step 2: Quantize weight（已验证于 002）

w_q = int8 per_channel(axis=0) of W
```
col0 amax=5.0: [1.0,3.0,5.0] → norm=[0.2,0.6,1.0] → int8=[0.203125,0.59375,1.0] → *5.0 = [1.015625, 2.96875, 5.0]
col1 amax=6.0: [2.0,4.0,6.0] → norm=[0.333,0.667,1.0] → int8=[0.328125,0.671875,1.0] → *6.0 = [1.96875, 4.03125, 6.0]
```
w_q = [[1.015625, 1.96875], [2.96875, 4.03125], [5.0, 6.0]]
