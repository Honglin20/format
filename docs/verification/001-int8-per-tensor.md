## 量化公式（int8 per_tensor）

int8 格式参数：
- ebits=0, mbits=8, max_norm = (2^7 - 1) / 2^6 = 127/64 = 1.984375

per_tensor 量化直接调用 `quantize_elemwise()`，不做 scale 归一化：

```
def _quantize_elemwise_core(A, bits=8, exp_bits=0, max_norm=1.984375, ...):
    # exp_bits=0 → private_exp = None
    out = A * 2^(bits-2) = A * 64         # _safe_lshift(×64)
    out = sign(out) * floor(|out| + 0.5)   # _round_mantissa(nearest)
    out = out / 64                          # _safe_rshift(/64)
    out = clamp(out, -1.984375, 1.984375)   # saturate_normals (ebits==0 → True)
```

**量化等级**: 127 个离散值: {-127/64, -126/64, ..., 0, ..., 126/64, 127/64}
**步长**: 1/64 = 0.015625
