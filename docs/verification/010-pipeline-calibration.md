## 验证内容

`MSEScaleStrategy.compute()` 的 grid search 算法正确性：对已知张量做 5-step 搜索，验证返回的 optimal scale 与手工计算一致。

## 算法

```
amax = max(|x|) along axis
candidates = linspace(0.5, 2.0, n_steps)  # [0.5, 0.875, 1.25, 1.625, 2.0]
for each factor in candidates:
    scale = amax * factor
    x_q = round(x/scale * 127) / 127 * scale   # _simple_quantize
    mse = mean((x - x_q)^2) along axis
    keep scale with lowest mse
return best_scale
```

```python

n_steps = 5
axis = 0  # per_channel along columns
```
