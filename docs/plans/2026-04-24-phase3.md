## 共性原则（贯穿 P3.1~P3.6）

### 1. 等价性门槛 = bit-exact

所有算子的 forward 和 backward 输出必须与 `mx/` 对应函数**完全按位相等**，即：

```python
assert torch.equal(src_out, mx_out)                              # forward
assert torch.equal(src_grad_input, mx_grad_input)                # backward
assert torch.equal(src_grad_weight, mx_grad_weight)              # backward
assert (src_grad_bias is None and mx_grad_bias is None) or \
        torch.equal(src_grad_bias, mx_grad_bias)                 # backward
```

**dither 舍入模式**：`torch.manual_seed(...)` 固定后 bit-exact。
**device/dtype**：默认 cpu + fp32，GPU 路径在 P3.6 做一次收口验证。
**numerical noise**：量化路径全是确定性算术，不允许 atol/rtol。

### 2. 算子接口契约（每个子阶段共用）

```python
class QuantizedXxx(ObservableMixin, nn.Xxx):
    def __init__(self, ..., cfg: OpQuantConfig, name: str | None = None):
        super().__init__(...)
        self.cfg = cfg
        self._analysis_name = name

class XxxFunction(torch.autograd.Function):
    @staticmethod
    def forward(ctx, ..., cfg: OpQuantConfig): ...
    @staticmethod
    def backward(ctx, grad_output): ...
```

- 所有算子继承 `ObservableMixin`（Phase 3 空骨架，Phase 4 启用）
- forward/backward 实现基于"for scheme in cfg.<role>: x = quantize(x, scheme)"的机械消费样板
- backward 路径做 QAT 量化；`cfg.grad_*` 全为空时退化为 STE（等同纯 inference）
